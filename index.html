PROJECTS


GASTOp

Genetic algorithm for structural design and topological optmization (GASTOp) is a Python package that provides a simple interface for optimizing truss structures subject to user-specified loading and boundary conditions. It employs a genetic algorithm to approach the "fittest" structure, where fitness is defined as a compromise between minimum factor of safety, maximum deflection, weight, and cost.

The package supports multiple crossover, mutation, and selection methods, as well as input file parsing, output plotting, and multithreading. Check out the GitHub repository or peruse the Read the Docs page!

Magnetic Nanoparticle Feedback Control System

Magnetic drug delivery is a more focused (think less harmful) alternative to intravenous chemotherapy, but current research in the field often overlooks practical obstacles like blood flow. My team and I designed an experimental setup to investigate the feasibility of two-dimensional magnetic nanoparticle control subject to oncoming flow. Our project was awarded The Mechanical Engineering Design Prize for best senior project.

We developed a system to control a nanoparticle cluster in a horizontal plane within a flow channel. I wrote a Python program to track the cluster’s position with computer vision and varied the strength and location of two electromagnets on rails using a PID control algorithm and Arduino. This enabled full control over the cluster's position both along and across the axis of the flow channel. See our journal-style paper for more info!

Turbojet Compressor

Design and simulation of a multi-stage turbojet compressor. Final project in my undergraduate Computational Fluid Dynamics (CFD) course.
